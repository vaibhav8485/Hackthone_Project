Create React App:

(1) npm install -g create-react-app = To Install App Globally
(2) npx create-react-app [app_name] = Create React App
(3) cd [app_name] = Go in App Dir.
(4) npm start = Start the App


Create React-Vite App:

(1) npm create vite@latest = Create React-Vite App
(2) cd [app_name] = Go in App Dir.
(3) npm install = Install all npm Pkg.
(4) npm run dev = To Run the React-Vite App
